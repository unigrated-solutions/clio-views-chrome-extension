# Clio Table Views Chrome Extension

This extension allows you to save and quickly apply custom column views for **Clio Contacts** and **Clio Matters** pages.

You can:
- Save which columns are visible
- Restore views instantly
- Keep your preferred layouts consistent
- Reopen filtered views using saved URL parameters

---

# Installation (Step-by-Step)

Follow these steps carefully. This only takes a few minutes.

## 1. Download the Extension

1. Click the green **Code** button on this page
2. Click **Download ZIP**
3. Save the file to your computer
4. Right-click the ZIP file and choose **Extract All**
5. Open the extracted folder

---

## 2. Open Chrome Extensions Page

1. Open Google Chrome
2. In the address bar, go to:

chrome://extensions

3. Turn ON **Developer Mode** (top right corner)

---

## 3. Install the Extension

1. Click **Load unpacked**
2. Select the folder you extracted earlier
3. Click **Select Folder**

✅ The extension should now appear in your list

---

## 4. Pin the Extension

1. Click the puzzle icon in Chrome (top right)
2. Find **Clio Table Views**
3. Click the 📌 (pin icon)

---

## 5. Open the Extension

1. Go to Clio (Contacts or Matters page)
2. Click the extension icon
3. The **side panel** will open

---

# How to Use

## Save a View

1. Go to a Clio page (Contacts or Matters)
2. Click **Refresh Columns**
3. Select the columns you want
4. Enter a name
5. Click **Save Current Selection**

---

## Apply a Saved View

1. Select a saved view from the dropdown
2. Click **Apply**

OR

Use the library section:
- Apply Contact Views
- Apply Matter Views

---

## Delete a View

1. Select the saved view
2. Click **Delete**
